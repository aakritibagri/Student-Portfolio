Featured Projects

Smart Agri Connect

Smart Agri Connect is an all-in-one mobile platform that empowers farmers by directly connecting them with customers and marketplaces, eliminating middlemen. Built using Java (Android), Python (AI/ML), and MongoDB, the app integrates e-commerce, social networking, and smart farming tools. Key features include customizable farmer profiles, inventory management, interactive maps for location-based selling, multilingual support, and AI-driven product recommendations. The platform promotes fair pricing, reduces waste, and enhances market access—creating a transparent and tech-driven agricultural ecosystem.

Grow Buddy

Grow Buddy is a dynamic web platform designed for hosting and participating in coding competitions and hackathons. Built using the MERN stack (MongoDB, Express.js, React.js, Node.js), Grow Buddy empowers students and developers to explore, engage, and grow through tech events like DSA contests, competitive programming, and machine learning challenges. The platform features event listings, easy registration, a gamified leveling system, social sharing, and a leaderboard to encourage active participation and community recognition.
